Length

A length tulajdonsággal meg tudjuk határozni egy tömb vagy string (szöveg) hosszát. Például a for ciklus használatában is jelentős szerepe van. Gyakran használjuk feltételeknél, ahol meg akarjuk vizsgálni, hogy a tömbnek vannak-e elemei, illetve hogy mennyi eleme van. Mindig integer (egész szám) értéket kapunk.

Példa

const fruits = ['alma', 'körte', 'szilva', 'barack', 'narancs'];

//Megkapjuk a tömb hosszát
console.log(fruits.length);

//feltétellel
if(fruits.length > 0){
  console.log(`A tömb nem üres. A tömbnek ${fruits.length} eleme van.`);
} else {
  console.log('A tömb üres.');
}

Kimenet:

A tömb nem üres. A tömbnek 5 eleme van.

Map

A map() az egyik leggyakrabban használt metódus. Sokszor alkalmazzuk tömbök esetében, hogy végig iteráljunk rajtuk és hozzáférjünk az elemekhez (minden elemen alkalmazunk egy fügvényt), illetve azok indexeihez, amivel a későbbiekben könnyebben dolgozhatunk. Szinte nélkülözhetetlen API hívások esetén, amikor is JSON formátumban kapjuk meg az adatokat, amiket objektumokká alakítunk. Minden esetben egy új tömböt kapunk vissza.

Példa:

const posts = [
  {
    id: 1,
    title: 'Javascript alapjai',
    body: 'Pellentesque scelerisque, est id vehicula bibendum, est tellus rutrum justo, et dignissim libero sapien et erat.',
  },
  {
    id: 2,
    title: 'PHP keretrendszerek',
    body: 'Vestibulum interdum venenatis nunc et sodales. Nunc et urna non ipsum tempor faucibus. Pellentesque nec neque luctus, vulputate lorem eget, aliquam nisl. Phasellus aliquet pharetra finibus.',
  },
];

posts.map((post) =>
console.log('id:', post.id, 'title:', post.title, 'body:', post.body)
);

Kimenet:

id: 1 title: Javascript alapjai body: Pellentesque scelerisque, est id vehicula bibendum, est tellus rutrum justo, et dignissim libero sapien et erat. id: 2 title: PHP keretrendszerek body: Vestibulum interdum venenatis nunc et sodales. Nunc et urna non ipsum tempor faucibus. Pellentesque nec neque luctus, vulputate lorem eget, aliquam nisl. Phasellus aliquet pharetra finibus.

Filter

A filter() metódust arra használjuk, hogy egy tömbből kiválogassuk azokat az elemeket, amelyek megfelelnek egy megadott feltételnek. Általa elérhetjük a tömb egyes elemeit és azok indexeit is. Gyakran használom egy elem törlésére, ahol az id alapján beazonosítom az elemet és annak kivételével jelenítem meg a tömböt (A példában is ez látható). Hasonlóan a map()-hez, itt is egy új tömböt kapunk vissza.

Példa:

const posts = [
  {
    id: 1,
    title: 'Javascript alapjai',
    body: 'Pellentesque scelerisque, est id vehicula bibendum, est tellus rutrum justo, et dignissim libero sapien et erat.',
  },
  {
    id: 2,
    title: 'PHP keretrendszerek',
    body: 'Vestibulum interdum venenatis nunc et sodales. Nunc et urna non ipsum tempor faucibus. Pellentesque nec neque luctus, vulputate lorem eget, aliquam nisl. Phasellus aliquet pharetra finibus.',
  },
];

const selectedPost = posts.filter((post) => post.id !== 1);
console.log(selectedPost);
);

Kimenet:

Array(1) 0 : {id: 2, title: 'PHP keretrendszerek', body: 'Vestibulum interdum venenatis nunc et sodales. Nun…aliquam nisl. Phasellus aliquet pharetra finibus.'} length : 1
